# Changelog

All notable changes to the ASG VS Code extension.

## [1.0.0] - 2025-01-31

### Added
- Syntax highlighting for ASG (.asg, .asg files)
- Language Server Protocol (LSP) integration
- Code snippets for common patterns
- Commands:
  - Run File (`Ctrl+Shift+R`)
  - Run Selection (`Ctrl+Enter`)
  - Start REPL
  - Restart LSP
- Configuration options for LSP
- Rainbow bracket matching support
- Auto-closing pairs for parentheses and strings
- Folding support for function definitions
